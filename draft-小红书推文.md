## 正文

每次 git commit 的时候，你是不是也这样👇

❌ update
❌ fix bug  
❌ 改了点东西
❌ wip
❌ 不知道写啥

然后被同事吐槽提交历史像日记🥲

我写了个小工具叫 gocommit，让 AI 帮你写中文 commit message！

👉 git add 之后，敲一个 `gocommit`

AI 自动分析你的代码变更，生成规范的中文提交信息：

```
feat: 添加JWT认证中间件
fix(http): 修复请求超时未重试的问题
refactor: 重构配置加载逻辑
```

比你自己写的还规范😭

---

✨ 三个亮点：

1️⃣ 中文优先！英文commit工具一堆，中文的几乎没有

2️⃣ 支持 OpenCode/Claude 的 Skill！装上之后在 AI 对话里说"帮我提交"就行了，连命令都不用敲

3️⃣ Go 写的，一个二进制文件，没有乱七八糟的依赖
